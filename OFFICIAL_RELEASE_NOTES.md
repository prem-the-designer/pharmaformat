## 📋 Executive Summary
Pharmat is an advanced AI-driven editorial assistant that automates the detection, formatting, and validation of pharmaceutical terminology to ensure 100% compliance with medical style guides.

---

## 🚀 Key Features
*Each feature is streamlined for maximum efficiency and clarity.*

*   **Intelligent Auto-Formatting**: Automatically converts detected brand names to their chemically accurate casing while preserving generic name standards.
*   **Precision Typos Correction**: Utilizes Levenshtein distance algorithms to identify and rectify spelling errors in drug names instantly.
*   **Dynamic Anomaly Detection**: Proactively highlights unknown or unverified terms to prevent publication errors.
*   **Context-Aware Tokenization**: Distinguishes between standard text and medical terminology to apply specific formatting rules without disrupting narrative flow.
*   **Centralized Dictionary Management**: Offers full CRUD capabilities for a collaborative, team-wide source of truth for drug terminologies.
*   **Bulk Data Integration**: Supports drag-and-drop xlxs import to effortlessly migrate and update massive drug databases.
*   **Seamless Google Verification**: Integrated one-click external search allows for rapid validation of unidentified pharmaceutical terms.
*   **Glassmorphism UI/UX**: Features a modern, accessible interface with fixed overlays and non-intrusive notifications for an optimized user experience.
